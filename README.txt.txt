🎓 Student Performance Dashboard (Excel)
Project Overview
Developed an interactive Excel dashboard to analyze student academic performance, attendance, and grade distribution. The dashboard provides insights into student success rates, score trends, and factors influencing academic outcomes.
Features
 Total Students KPI
 Average Final Score KPI
 Total Passed KPI
 Pass Rate Percentage KPI
 Grade Distribution Analysis
 Pass vs Fail Comparison
 Parent Education Impact Analysis
 Attendance vs Score Correlation
 Interactive Slicers for:
Grade
Pass/Fail Status
Parent Education Level
Study Environment
Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Data Analysis
Key Insights
Achieved a 93% overall pass rate.
Analyzed grade distribution across 1,000 students.
Identified relationships between attendance and academic performance.
Evaluated the impact of parental education and study environment on student scores.
Enabled dynamic filtering for interactive exploration of student data.
