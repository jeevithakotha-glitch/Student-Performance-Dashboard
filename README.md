🎓 Student Performance Dashboard (Excel)

 Project Overview

The Student Performance Dashboard is an interactive Excel-based analytics project designed to evaluate student academic performance, attendance patterns, and grade distribution. The dashboard provides meaningful insights into student outcomes and helps identify factors that influence academic success through dynamic visualizations and KPIs.

---

 Features

 Key Performance Indicators (KPIs)

-  Total Students
-  Average Final Score
-  Total Passed Students
-  Pass Rate Percentage

 Dashboard Analysis

- Grade Distribution Analysis
- Pass vs Fail Comparison
- Parent Education Impact Analysis
- Attendance vs Score Correlation

 Interactive Filters

- Grade
- Pass/Fail Status
- Parent Education Level
- Study Environment

---

 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Visualization Techniques

---

📈 Key Insights

- Achieved an overall 93% Pass Rate.
- Analyzed academic performance data for 1,000 students.
- Identified a positive relationship between attendance and final scores.
- Evaluated the influence of parental education levels on student performance.
- Explored the impact of different study environments on academic outcomes.
- Enabled dynamic analysis through interactive slicers and filters.
